# Hex Game

The game of Hex is an abstract combinatorial board game for two players.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install Hex Game.

```bash
npm i
```

## Usage

```bash
# To lauch the app
node app.js
```
